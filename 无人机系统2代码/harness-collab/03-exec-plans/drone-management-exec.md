## 4. 测试计划

### 4.1 测试覆盖范围

| 模块 | 测试类型 | 测试内容 |
|------|----------|----------|
| Controller | 单元测试 | 参数校验、异常处理、响应格式 |
| Service | 单元测试 | 业务逻辑、数据转换 |
| Mapper | 集成测试 | SQL正确性、分页查询 |
| API | 接口测试 | CRUD操作、权限控制 |
| 数据库 | 集成测试 | SQLite/MySQL切换 |
| 前端 | UI测试 | 页面渲染、交互功能 |

### 4.2 测试用例设计

#### 4.2.1 无人机列表查询

| 测试场景 | 预期结果 |
|----------|---------|
| 正常查询（无参数） | 返回第一页数据 |
| 分页查询（page=2, size=5） | 返回第二页数据，每页5条 |
| 按名称模糊查询 | 返回匹配的无人机列表 |
| 按状态筛选 | 返回指定状态的无人机 |

#### 4.2.2 无人机详情查询

| 测试场景 | 预期结果 |
|----------|---------|
| 查询存在的无人机 | 返回无人机详情 |
| 查询不存在的无人机 | 返回404错误 |

#### 4.2.3 新增无人机

| 测试场景 | 预期结果 |
|----------|---------|
| 正常新增（参数完整） | 创建成功，返回新ID |
| 缺少必填参数 | 返回400错误，提示缺少字段 |
| 参数不符合约束（weight≤0） | 返回400错误，提示参数校验失败 |

#### 4.2.4 更新无人机

| 测试场景 | 预期结果 |
|----------|---------|
| 正常更新 | 更新成功 |
| 更新不存在的无人机 | 返回404错误 |
| 更新参数不符合约束 | 返回400错误 |

#### 4.2.5 删除无人机

| 测试场景 | 预期结果 |
|----------|---------|
| 删除存在的无人机 | 删除成功 |
| 删除不存在的无人机 | 返回404错误 |
| 普通用户尝试删除 | 返回403错误 |
